# RESTful API Example with Spring Boot, Spring Data REST and MySQL

## Guide
https://github.com/hellokoding/restfulapi-springboot-springdatarest-mysql

## Prerequisites
- JDK 1.7+
- Maven 3+
- MySQL 5.6+

## Stack
- Spring Boot
- Spring Data JPA
- Spring Data REST
- MySQL

## Run
`mvn clean spring-boot:run`
