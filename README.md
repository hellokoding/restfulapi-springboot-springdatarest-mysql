# RESTful API Example with Spring Boot, Spring Data REST and MySQL

## Guide
https://hellokoding.com/restful-api-example-with-spring-boot-spring-data-rest-and-mysql/

## Prerequisites
- JDK 1.7+
- Maven 3+
- MySQL 5.6+

## Stack
- Spring Boot
- Spring Data JPA
- Spring Data REST
- MySQL

## Run
`mvn clean spring-boot:run`
